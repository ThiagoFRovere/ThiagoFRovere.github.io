# Thiago Ferreira Rôvere | Portfólio

Portfólio profissional desenvolvido para apresentar minha trajetória, competências técnicas e projetos em desenvolvimento web, automação e infraestrutura.

O site foi construído com foco em apresentação profissional, navegação clara e exposição de projetos reais publicados ou desenvolvidos para uso comercial, institucional e interno.

## Acesso

- Site principal: `index.html`
- Versão em inglês: `index_english.html`

## Sobre o projeto

Este repositório reúne meu portfólio pessoal publicado no GitHub Pages / domínio próprio, com seções voltadas para:

- apresentação profissional
- competências técnicas
- trajetória e experiências
- vitrine de projetos
- canais de contato

Além da página principal, o repositório também contém subprojetos e demonstrações vinculadas dentro da seção de projetos.
